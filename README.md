# Alke Wallet - Proyecto Billetera Digital

Este proyecto consiste en el desarrollo de una interfaz de billetera virtual funcional para el modulo de Fundamentos de Desarrollo Frontend. La aplicacion permite gestionar operaciones financieras basicas mediante una interfaz web responsiva.

# Funcionalidades Principales

Acceso de usuario: Pantalla de inicio de sesion funcional.
Visualizacion de saldo: Dashboard principal que muestra el monto disponible actualizado en tiempo real.
Gestion de fondos: Modulo para realizar depositos y retiros de dinero que impactan directamente en el saldo.
Transferencias: Sistema de envio de dinero a contactos con validacion de saldo disponible.
Historial de movimientos: Registro detallado de todas las operaciones realizadas durante la sesion.

# Tecnologias Utilizadas

HTML5: Estructura de las paginas y formularios.
CSS3: Estilos personalizados para la apariencia visual.
Bootstrap 5: Framework utilizado para el dise√±o responsivo, botones y ventanas modales de confirmacion.
jQuery: Libreria para la manipulacion del DOM y manejo de eventos de usuario.
JavaScript: Logica para el calculo de saldos y procesamiento de datos.
LocalStorage: Metodo de almacenamiento en el navegador para que los datos no se borren al refrescar la pagina.

# Instrucciones de Uso

1. Clonar el repositorio o descargar los archivos.
2. Abrir el archivo index.html en cualquier navegador moderno.
3. Iniciar sesion para acceder al menu principal.
4. Utilizar las opciones de deposito y retiro para gestionar el saldo inicial.
5. Acceder al historial para verificar los movimientos registrados.

# Estructura de Archivos

index.html: Punto de entrada a la aplicacion.
login.html: Interfaz de acceso.
menu.html: Panel principal de usuario.
deposit.html: Seccion de carga y retiro de fondos.
sendmoney.html: Formulario de envio a contactos.
transactions.html: Tabla de historial.
script.js: Contiene toda la logica funcional en JavaScript y jQuery.
style.css: Hoja de estilos complementaria a Bootstrap.
